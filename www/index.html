    Хеш пароля для basic auth

    Введите пароль — получите md5-хеш $1$… для переменной окружения BASIC_AUTH_HASH контейнера.

    Логин задаётся в BASIC_AUTH_USER (по умолчанию admin). По умолчанию пароль — admin.

    AWG / WireGuard .conf → YAML

    Вставьте содержимое .conf (AmneziaWG или обычный WireGuard) — получите блок прокси type: wireguard для секции proxies.

    TrustTunnel .toml → YAML

    Вставьте TrustTunnel-конфиг .toml — получите блок прокси type: trusttunnel для секции proxies.

    OpenVPN .ovpn → YAML

    Вставьте OpenVPN-конфиг (со встроенными блоками <ca>/<cert>/<key> или auth-user-pass) — получите блок прокси type: openvpn.
